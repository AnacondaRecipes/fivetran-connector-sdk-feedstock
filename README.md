Installing fivetran-connector-sdk
==================

Installing `fivetran-connector-sdk` from the main channel can be achieved by:

```
conda install fivetran-connector-sdk
```

It is possible to list all of the versions of `fivetran-connector-sdk` available on your platform with `conda`:

```
conda search fivetran-connector-sdk
```
